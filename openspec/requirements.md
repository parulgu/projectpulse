# Project Pulse Requirements

## Capability: Project Management

Manage projects, members, classifications, navigation, and project lifecycle.

## Capability: Action Management

Track, update, filter, and review project actions.

## Capability: Meeting Notes

Capture notes, upload AI companion notes, and review extracted actions.

## Capability: AI Extraction

Generate actions from meeting notes using OCI Generative AI and fallback extraction logic.

## Capability: Project Memory

Track decisions, answer project questions, and generate executive summaries.

## Capability: Milestones And Planning

Manage milestone phases, phase items, and useful links.

## Capability: Bug Database

Import, refresh, filter, and analyze project bug information.

## Capability: Backend Services

Persist and expose project data through APIs.
